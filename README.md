# 🍎 Smart Lesson Plan System

A premium, cross-platform (Web & Mobile) digital notebook designed specifically for Filipino teachers, strictly following the **DepEd Detailed Lesson Plan (DLP)** format (DepEd Order No. 42 s. 2016).

---

## 🚀 Live Access
- **Firebase Demo (Primary)**: [https://smart-lesson-plan-app.web.app](https://smart-lesson-plan-app.web.app)
- **GitHub Pages (Backup)**: [https://Ianroy18.github.io/smart-lesson-plan-app/](https://Ianroy18.github.io/smart-lesson-plan-app/)

---

## 📱 Download Android App
You can download the latest **APK** for your Android phone directly from the **Actions** tab or **Releases** section of this repository.

1. Go to the [Actions Tab](https://github.com/Ianroy18/smart-lesson-plan-app/actions).
2. Click on the latest successful "Deploy and Build APK" run.
3. Scroll down to **Artifacts** and download `release-apk`.

---

## ✨ Features
- **DepEd Standard Compliance**: Multi-step builder for Objectives, Content, Resources, Procedures (A-J), Assessment, and Reflection.
- **Offline-First**: Save your work locally using SQLite (Mobile) or LocalStorage (Web).
- **Professional PDF Export**: Generate ready-to-print PDFs with "by kuya ian" branding.
- **Built-in Email**: Send lesson summaries directly via email.
- **Firebase Auth**: Secure login and account management.
